# Prompts 沉淀目录

本目录用于存放用户与 Claude 交互的 Prompt 整理，目的是将高效表达模式沉淀为长期资产。

---

## 目录结构

```
prompts/
├── daily/           # 每日 Prompt 整理
│   └── 2026-04-01.md
│   └── ...
│
├── patterns/        # 表达模式（长期资产）
│   ├── analysis.md  # 分析类模式
│   ├── creation.md  # 创作类模式
│   ├── decision.md  # 决策类模式
│   └── ...
│
└── templates/       # 可复用模板
    └── ...
```

---

## 每日整理格式

`prompts/daily/{date}.md` 格式：

```markdown
# Prompt 整理 - {date}

## 模式1：{模式名称}

**原始 Prompt**：
> 用户说的原话

**上下文**：
- 场景：xxx
- 目的：xxx

**效果评估**：
- 是否高效：是/否
- 可复用性：高/中/低

**提炼模式**：
{抽取出的通用表达}

---

## 模式2：...
```

---

## 模式沉淀标准

什么样的 Prompt 有资格进入 `patterns/`：

1. **高效**：能快速达成目标
2. **可复用**：不依赖特定上下文
3. **有启发**：表达方式有学习价值
4. **已验证**：多次使用效果稳定

---

## 使用方式

Agent 启动时：
1. 读取最近 7 天的 daily
2. 根据任务类型召回相关 patterns
3. 应用模式辅助用户

---

## 维护规则

- 每日整理：当天对话结束后
- 模式提取：每周从 daily 中筛选
- 模式更新：相似模式合并，冗余模式归档
- Git 记录：每次变更留痕
