# 智汇投研 - 基于多智能体辩论的AI投研助手

## 项目简介

智汇投研是一个创新性的AI投研平台，通过**多智能体协作**和**红蓝军辩论机制**，为个人投资者提供专业的股票分析服务。

### 核心创新点

1. **四维分析师团队**: 基本面、技术面、情绪面、新闻面四位分析师并行分析
2. **红蓝军辩论室**: 看多vs看空研究员的结构化辩论，让用户看到观点碰撞
3. **智能决策引擎**: 交易员→投资组合经理的完整决策流程
4. **白盒化设计**: 将Agent协作过程透明展示，增强可解释性

## 快速开始

### 1. 环境准备

```bash
# 克隆项目
cd fintech_retail_assistant

# 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows

# 安装依赖
pip install -r requirements.txt
```

### 2. 配置API密钥

```bash
# 复制配置模板
cp .env.example .env

# 编辑 .env 文件，填入你的API密钥
# 至少配置一个LLM提供商的密钥
```

### 3. 运行

**命令行模式:**
```bash
python main.py
```

**Streamlit界面:**
```bash
streamlit run ui/app.py
```

## 核心功能

### 1. 多维度分析看板
- **基本面分析**: 财务指标、估值评估、红旗警告
- **技术面分析**: MA均线、RSI、MACD、布林带
- **情绪面分析**: 市场情绪评分、逆向信号识别
- **新闻面分析**: 重大事件、政策影响、催化剂识别

### 2. 红蓝军辩论室
- 看多研究员挖掘上涨理由
- 看空研究员警示风险隐患
- 多轮辩论，观点交锋
- 用户可看到完整辩论过程

### 3. 智能决策引擎
- 交易员综合建议
- 投资组合经理风险审核
- 仓位管理建议
- 止损止盈策略

## 支持的LLM

- OpenAI (GPT-4o)
- 智谱AI (GLM-4)
- DeepSeek
- 阿里通义千问 (Qwen)
- Ollama (本地部署)

## 技术栈

- **框架**: LangGraph (Agent工作流编排)
- **数据源**: akshare (A股数据)
- **前端**: Streamlit
- **LLM集成**: LangChain

## 项目亮点（面试向）

1. **产品设计创新**: 将Agent黑盒流程白盒化，辩论机制增强可解释性
2. **技术深度**: 理解并实践Multi-Agent协作、LangGraph工作流编排
3. **工程能力**: 模块化设计、支持多LLM、完整的错误处理
4. **用户视角**: 考虑风险偏好、提供分级决策、完整的用户旅程
