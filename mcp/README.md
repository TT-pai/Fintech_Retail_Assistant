# MCP 目录

本目录存放 MCP 服务配置，**克制使用**。

---

## 使用原则

1. **优先 CLI**：能用 CLI 的不用 MCP
2. **优先脚本**：能用 Bun 脚本的不用 MCP
3. **必要时引入**：CLI 和脚本都无法满足时才考虑 MCP

---

## 可能需要的 MCP

如果确实需要，优先考虑：

- **filesystem**：文件系统操作（但 CLI 已足够）
- **fetch**：网络请求（但 Bun 已足够）

---

## 配置方式

如果引入 MCP，配置放在：

```
mcp/
├── config.json
├── servers/
└── logs/
```
